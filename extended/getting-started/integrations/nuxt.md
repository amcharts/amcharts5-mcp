---

Follow the directions for [installing Nuxt 3](https://v3.nuxtjs.org/getting-started/installation/).

Then create a `components/Chart.vue` file which contains the Vue code from our [Vue tutorial](https://www.amcharts.com/docs/v5/getting-started/integrations/vue/).

Lastly, you can use the Chart component like this:

<ClientOnly>
  <Chart />
</ClientOnly>
