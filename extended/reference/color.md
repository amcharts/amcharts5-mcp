Wherever color needs to be specified in amCharts 5, Color object needs to be used.

## Import

```javascript
// Import Color
import * as am5 from "@amcharts/amcharts5"
```

## Properties

- **b** (`number`) — Value of color's B channel.
- **g** (`number`) — Value of color's G channel.
- **hex** (`number`) — Color numeric value.
