Draws a rectangle.

## Import

```javascript
// Import Rectangle
import * as am5 from "@amcharts/amcharts5"
```

## Inheritance

Extends: Graphics
Extended by: RoundedRectangle

> **Note:** This class also inherits all settings, properties, methods, and events from Graphics (and its ancestors). Use `get_doc` or `get_core_reference` with the parent class name to see inherited members.

## Settings

- **containStroke** (`undefined | false | true`) — If this is set to true, rectangle will be drawn in such a way that its stroke is contained within the rectangle's width and height.
