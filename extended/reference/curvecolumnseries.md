A column series for use in a CurveChart, SerpetineChart, or a SpiralChart.

## Import

```javascript
// Import CurveColumnSeries
import * as am5timeline from "@amcharts/amcharts5/timeline"
```

## Inheritance

Extends: BaseColumnSeries

> **Note:** This class also inherits all settings, properties, methods, and events from BaseColumnSeries (and its ancestors). Use `get_doc` or `get_core_reference` with the parent class name to see inherited members.

## Properties

- **chart** (`CurveChart | undefined`) — A chart series belongs to.
- **columns** (`ListTemplate`) — Default new ListTemplate<Polygon> A TemplateList of all columns in series. columns.template can be used to set default settings for all columns, or to change on existing ones.
