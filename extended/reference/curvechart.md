Base chart for a Timeline chart.
 For this chart to work, it needs curve points provided via renderer of its X-axis.
 Note: it is an experimental chart type and does not support all the functionality of the XYChart.

## Import

```javascript
// Import CurveChart
import * as am5timeline from "@amcharts/amcharts5/timeline"
```

## Inheritance

Extends: XYChart
Extended by: SerpentineChart, SpiralChart

> **Note:** This class also inherits all settings, properties, methods, and events from XYChart (and its ancestors). Use `get_doc` or `get_core_reference` with the parent class name to see inherited members.
