A date axis that is used as an X (horizontal) axis for Gantt charts.

## Import

```javascript
// Import GanttDateAxis
import * as am5gantt from "@amcharts/amcharts5/gantt"
```

## Inheritance

Extends: DateAxis

> **Note:** This class also inherits all settings, properties, methods, and events from DateAxis (and its ancestors). Use `get_doc` or `get_core_reference` with the parent class name to see inherited members.

## Properties

- **gantt** (`Gantt`) — A reference to the parent Gantt chart.
