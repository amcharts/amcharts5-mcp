## Properties

- **key** (`string`) — Setting key.
- **minValue** (`undefined | number`) — Minimum numeric value allowable in this field. @since 5.8.1
- **name** (`string`) — Name of the setting (displayed in edit modal). Settings with the same name will be grouped in modal.
- **options** (`Array`) — If type: "dropdown", options should contain a list of items it.
