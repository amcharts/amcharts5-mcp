## Inheritance

Extends: ISeriesDataItem
Extended by: IStockLegendDataItem

> **Note:** This class also inherits all settings, properties, methods, and events from ISeriesDataItem (and its ancestors). Use `get_doc` or `get_core_reference` with the parent class name to see inherited members.

## Properties

- **fill** (`Color`) — Marker fill color.
- **itemContainer** (`Container`) — Container element holding all other legend item elements, labels, markers, etc.
- **label** (`Label`) — Label element.
- **marker** (`Container`) — Marker element.
- **markerRectangle** (`RoundedRectangle`) — Marker rectangle element.
- **name** (`undefined | string`) — Name of the legend item.
- **stroke** (`Color`) — Marker stroke (outline) color.
- **valueLabel** (`Label`) — Value label element.
