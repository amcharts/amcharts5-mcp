Draws a line series for use in a RadarChart.
Data item
RadarLineSeries uses data items of type IRadarLineSeriesDataItem.

## Import

```javascript
// Import RadarLineSeries
import * as am5radar from "@amcharts/amcharts5/radar"
```

## Inheritance

Extends: LineSeries
Extended by: SmoothedRadarLineSeries

> **Note:** This class also inherits all settings, properties, methods, and events from LineSeries (and its ancestors). Use `get_doc` or `get_core_reference` with the parent class name to see inherited members.

## Settings

- **connectEnds** (`undefined | false | true`) — Default @true If set to true (default), series will connect its last data point to the first one with a line, thus completing full circle. Click here for more info

## Properties

- **chart** (`RadarChart | undefined`) — A chart series belongs to.
