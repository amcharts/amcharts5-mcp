A pre-defined period selector control for StockToolback.

## Import

```javascript
// Import PeriodSelector
import * as am5stock from "@amcharts/amcharts5/stock"
```

## Inheritance

Extends: StockControl

> **Note:** This class also inherits all settings, properties, methods, and events from StockControl (and its ancestors). Use `get_doc` or `get_core_reference` with the parent class name to see inherited members.

## Settings

- **hideLongPeriods** (`undefined | false | true`) — Default false Hide periods that are longer than the actual data. @since 5.3.9
- **periods** (`IPeriod[]`) — A list periods to choose from.
